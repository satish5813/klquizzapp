Place your two logo images here:

  logo-left.png    → shown on the LEFT of the header (e.g. university logo)
  logo-right.png   → shown on the RIGHT of the header (e.g. department / AI logo)

PNG with a transparent background works best. They are auto-scaled to ~44px tall.
If a file is missing, the header shows a small text badge instead (KL / AI) so the
layout never breaks. SVG also works — then reference /logo-left.svg in src/App.tsx.
